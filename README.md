# 🎨 AI Image Generator

A simple AI Image Generator web application built using **HTML, CSS, and JavaScript**.  
This app allows users to generate images from text prompts using an AI image generation API.

The user enters a description of the image, and the AI generates an image based on the prompt.

---

# 🚀 Features

- Generate images from text prompts
- Simple and clean user interface
- Fast image generation
- Responsive design
- Built with pure JavaScript

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- AI Image Generation API

# ⚙️ How to Use

1. Enter a text prompt describing the image you want.
2. Click the **Generate** button.
3. The AI will generate an image based on your prompt.
